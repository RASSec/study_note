## 查看经过位操作后可获得字符

```php
$a="0123456789abcdefghijklmnopqrstuvwxyz";
$ope="^|&";
for($i=0;$i<strlen($a);$i++)
{
    for($j=0;$j<strlen($a);$j++)
    {   
        for($k=0;$k<strlen($ope);$k++)
        {   
            $tmp="'".$a[$i]."'".$ope[$k]."'".$a[$j]."';";
            echo $tmp."     ";
            eval("echo ".$tmp);
            echo "<br />";
        }
    }
}
for($j=0;$j<strlen($a);$j++)
{   
 
    $tmp="~'".$a[$j]."';";
    echo $tmp."     ";
    eval("echo ".$tmp);
    echo "<br />";
}
```
